To-Do List - C++ Project
Description
Welcome to the To-Do List project, developed as part of the C++ course offered by Technohack Edutech. This project aims to create a simple command-line to-do list application using C++ programming language. The To-Do List allows users to manage their tasks effectively by adding, viewing, marking as completed, and removing tasks.

Features
Add Task: Add a new task to the to-do list along with a description and due date.

View Tasks: Display all the tasks in the to-do list, including their descriptions and due dates.

Mark Completed: Mark a task as completed once it's finished.

Remove Task: Remove a task from the to-do list if it's no longer needed.

Save and Load: The to-do list supports saving tasks to a file and loading tasks from a file, enabling persistent storage.

Error Handling: The application includes robust error handling to prevent crashes and unexpected behavior
